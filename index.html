Tempurature-responsive Fan

C++
Arduino

The system automatically turns the fan on when the temputature reach 20°C. It includes a manual override button, allowing users to turn the fan off when desired.

Click on the picture to see how to use it.

Working process

  • Started learning how to use an Arduino ESP32 and a DHT11 sensor to create a simple system that responds to temperature changes.
  • Worked on programming the fan to automatically turn on when the temperature reaches a set threshold, and I included a manual button to let users turn it off whenever they want.
  • Navigated challenges in understanding how to connect the hardware components and troubleshoot the system when it didn't work as expected.
  • Gradually developed better skills in coding and circuit design, especially in managing real-time data from sensors and adjusting the system based on feedback.
  • Gained a basic understanding of how to control small systems like this and learned a lot through trial and error, improving both my technical knowledge and confidence.

Working process

  • Started learning Python to build a chatbot that could help with my IELTS speaking practice when I didn't have a partner to practice with.

  • Faced challenges with integrating speech output and refining the randomization process, but gradually improved my coding skills through trial and error.
